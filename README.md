# 🏆 WinLog

A daily wins journal — capture achievements before you forget them.

**[Try it →](https://magic-dev-kz.github.io/winlog/)**

## Features
- Voice input — log wins hands-free
- Charts and streaks to visualize progress
- Mood tracker tied to your achievements
- Search, filter & pin your best wins
- Streak achievements to keep you motivated
- Dark / light mode

## Why WinLog?
We remember failures but forget wins. WinLog is a micro-journal that takes seconds to fill in — voice or text — and turns your daily achievements into charts, streaks, and pinned highlights you can revisit when you need a confidence boost.

## Tech
- Single HTML file — nothing to install
- Works offline (PWA)
- No accounts, no tracking
- Your data stays in your browser (localStorage)
- Free forever
